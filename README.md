# HTML Resume Page Assignment

This repository contains HTML Page Resume Assignment for ChaiCode

## How to setup

- Step 1: clone the github repository using below command:

  ```bash
  git clone https://github.com/akshat-raj09/html-resume-page-assignment.git
  ```

- Step 2: Navigate inside the project directory using below command or by double clicking on the project folder:
  ```bash
  cd html-resume-page-assignment/
  ```
- Step 3: Open the file `resume.html` in any web browser of your choice by double clicking on the file. This will open the HTML page in your system.
